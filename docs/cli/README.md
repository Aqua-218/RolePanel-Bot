# CLI Specification

Version: 1.0.0
Last Updated: 2026-01-08

## Overview

Bot provides slash commands for panel management and configuration.
All commands are guild-scoped (not global).

## Command List

| Command | Description | Permission |
|---------|-------------|------------|
| /panel create | Create new role panel | Manage Roles |
| /panel list | List all panels | Manage Roles |
| /panel edit | Edit existing panel | Manage Roles |
| /config audit-channel | Configure audit log channel | Administrator |
| /config show | Show current configuration | Administrator |
