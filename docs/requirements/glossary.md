# Glossary

Version: 1.0.0
Last Updated: 2026-01-08

## Terms

### Panel
ロールパネル。ユーザーがロールを選択するためのインタラクティブなメッセージ。
ボタンまたはセレクトメニューで構成される。

### Panel Item
パネル内の個々のロール選択肢。ロールID、ラベル、絵文字、説明で構成される。

### Panel Style
パネルの表示形式。Button（ボタン式）またはSelectMenu（セレクトメニュー式）。

### Ephemeral Message
送信者本人にのみ表示される一時的なメッセージ。他のユーザーには見えない。

### Audit Log
ロール付与・削除の履歴を管理者専用チャンネルに送信するログ機能。

### Guild
Discordサーバー。Discord APIでの正式名称。

### Interaction
ユーザーがボタンクリック、セレクトメニュー選択、スラッシュコマンド実行などを
行った際に発生するイベント。

### Shard
Discord Gatewayへの接続単位。大規模Botでは複数シャードで負荷分散する。
本プロジェクトは個人サーバー用のため単一シャードで運用。

### Component
Discord メッセージに添付できるインタラクティブ要素。
ボタン、セレクトメニュー、テキスト入力などがある。

### Modal
ユーザー入力を受け付けるポップアップダイアログ。
テキスト入力フィールドを含むことができる。

### Custom ID
コンポーネントを識別するための文字列。最大100文字。
インタラクション発生時にこのIDで処理を振り分ける。
